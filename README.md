# Data Structures & Algorithms

This repository documents my journey of solving Data Structures and Algorithms problems, primarily on **LeetCode**, using **C++**.

The objective isn't just to collect accepted solutions, but to build strong pattern recognition, maintain reusable templates, and record insights from problems that introduce new ideas or techniques.

---

## Repository Structure

```text
.
├── Problems/      # LeetCode solutions
├── Templates/     # Reusable algorithm implementations
├── Notes/         # Pattern-based observations and revision notes
└── Resources/     # References and study material
```
